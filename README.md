## 🚀 Featured Projects

### 🍃 [Tea Leaf Disease Detection](https://github.com/NandhanaShanmugam/tea-leaf-disease-detection)
CNN-based multi-class disease classifier achieving **94% accuracy** on tea leaf images. Built with TensorFlow/Keras using transfer learning on MobileNetV2, with image augmentation and a two-phase fine-tuning strategy. Classifies 8 disease categories to help farmers catch crop issues early.

`Python` `TensorFlow` `Keras` `MobileNetV2` `Transfer Learning`

---

### 🛒 [B2B E-Commerce Platform](https://github.com/NandhanaShanmugam/ecommercewebsite)
Full-stack B2B marketplace with 4 user roles (Buyer, Seller, Logistics, Admin). Integrates **Hyperledger Fabric** permissioned blockchain for immutable order records. Built with Next.js + Node.js REST API, Java chaincode, CouchDB, Docker/Kubernetes orchestration, and AWS S3 for media storage.

`Next.js` `Node.js` `Hyperledger Fabric` `Docker` `Kubernetes` `AWS S3`

---

### 🎙️ Voice-Based Email System for Visually Impaired Users
Accessibility-first voice email client enabling fully hands-free email management. Achieved **87% command accuracy** using an NLP pipeline (NLTK) for intent classification — read, send, reply, delete, search — with fuzzy matching and accent robustness. Secured via OAuth2 + Gmail IMAP/SMTP, powered by Google Speech-to-Text and gTTS.

📄 *Published in the International Journal of Research and Analytical Reviews (IJRAR)*

`Python` `NLTK` `OAuth2` `Google Speech-to-Text` `gTTS` `IMAP/SMTP`
